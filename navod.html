Manual for custom-select tag

Files to include:

  • select.js
  • select.css

Updating the HTML:

  • Header:

    <script src="select.js" type="text/javascript"></script>

  • Select Definiton Example

    <select class="custom-select" id="s2" mode="2"> <option value="a1" title='Lorem ipsum1'>a1</option> <option value="a2" title='Lorem ipsum2'>a2</option> <option value="a3" title='Lorem ipsum3'>a3</option> <option value="a4" title='Lorem ipsum4'>a4</option> </select>

  • Mandatory TAG Attributes

    • class="custom-select"
    • id="s2" custom-select will not work properly without unique id

  • Other TAG Attributes

    • mode='1' -> Classic select without typing filter
    • mode='2' -> Select with typing filter

  • Option TAG Attributes

    • title='1' -> Additional text that will be included once select clicked
    • value='1' -> Unique identifier of the selected value
